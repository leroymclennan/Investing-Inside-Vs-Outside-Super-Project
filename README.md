# Investing-Inside-Vs-Outside-Super-Project
Python project with a formal report analyzing two investment strategies—inside super versus outside super. Users can run simulations under different assumptions and explore how factors like tax, fees, and liquidity impact long-term portfolio outcomes.

This project was undertaken following recent readings on the presence of 'tax drag' within superannuation funds, which may explain why fees alone fail to account for the difference in performance of the funds components versus performance of the fund. 
The concept of tax drag challenged my belief that investing in superannuation is superior to other investment vehicles given the tax concessions, as there must be a point in which the benefits of the tax concessions is outweighed by the drawbacks of fees and drag.
This project aimed to compare outcomes for investing surplus cashflow in superannuation versus outside superannuation, with a lesser goal of identifying the optimal strategy for allocating surplus cash flow between each vehicle.

This project additionally sought to develop my ability to use graphical user interfaces (GUI) for future projects. A simple GUI was utilised to allow the user to input their specific financial characteristics repeatedly, and compare between the resulting graphs.
